# Brainwave.io — Agency Landing Page

A clean, responsive agency landing page built with **pure HTML & CSS** — no JavaScript frameworks, no UI libraries, no dependencies.

## Live Preview

> Add your GitHub Pages or deployment link here

## Features

- Fully responsive layout
- Built with HTML5 and CSS3 only
- No frameworks or JavaScript dependencies
- Sections include: Hero, Services, Testimonials, About/Stats, Why Choose Us, Case Studies, and Footer
- Clean navigation with CTA buttons
- Multi-column footer with links

## Sections

| Section | Description |
|---|---|
| Navbar | Logo, nav links, and CTA button |
| Hero | Headline, subtext, and CTA |
| Services | Graphic Design, Web Development, Content Writing cards |
| Testimonial | Single quote block with avatar |
| Our Story | Stats (1M+ customers, 92% satisfaction, 4.9/5.0 rating) |
| Why Choose Us | Four feature highlights with icons |
| Case Studies | Portfolio grid with project cards |
| Footer | Multi-column links + branding |

## Getting Started

Clone the repo and open `index.html` in your browser — no build step required.

```bash
git clone https://github.com/your-username/brainwave-agency-landing.git
cd brainwave-agency-landing
open index.html
```
